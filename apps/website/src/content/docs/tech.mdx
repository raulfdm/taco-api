---
title: "Tecnologias"
---

- [NodeJS](https://nodejs.org/en) - JavaScript runtime para o Prisma conseguir gerar os arquivos (isso provavelmente é temporário)
- [Bun](https://bun.sh/) - JavaScript runtime para rodar os scripts e subir o servidor
- [ExpressJS](https://expressjs.com) - Framework HTTP
- [Prisma](https://www.prisma.io/) - Object-Relational Mapping (ORM) em JavaScript para popular o banco de dados e definir as relações
- [SQLite](https://sqlite.org/) - Mini banco de dados relacional
- [GraphQL](https://graphql.org/) - Linguagem de consulta para a API
- [GraphiQL](https://github.com/graphql/graphiql) - API playground (disponível apenas localmente em modo de desenvolvimento)
- [Starlight](https://starlight.astro.build/) - Gerador de site para documentação
