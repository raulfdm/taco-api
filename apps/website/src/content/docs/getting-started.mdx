---
title: "Començando"
---

Você pode tanto fazer um fork deste projeto quanto cloná-lo, fica a seu critério.

Antes de mais nada, garanta que você tenha instalado na sua máquina duas coisas:

- NodeJS versão 18 ou superior;
- BunJS versão 1.0.15 ou superior.

Agora, o primeiro passo é instalar as dependências do projeto:

```bash
bun install
```

Como o projeto está configurado como monorepo, para facilitar o desenvolvimento, sugiro navegar dentro da pasta da API:

```bash
cd apps/taco
```

Por último, rode o servidor de desenvolvimento com o comando:

```bash
bun run dev
```

Feito isso, a API estará rodando em `http://localhost:4000/graphql`.
