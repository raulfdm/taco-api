---
title: "Deploy como aplicação NodeJS"
---

Existem várias maneiras e várias plataformas para fazer o deploy desta API.

Em qualquer plataforma, procure por uma máquina que tenha suporte para aplicações NodeJS.

Apesar do projeto usar Bun, em uma máquina que tenha Node, você pode instalar Bun via npm:

```bash
npm add -g bun@1.0.18
```

Instalado o Bun globalmente, agora você pode seguir os passos normais:

Primeiro, instale as dependências:

```bash
bun install --production
```

Em seguida, na raiz do projeto, rode o comando:

```bash
bun run start --filter=taco-api
```

Isso vai fazer o build e executar o servidor.

Lembrando que a porta que precisa estar aberta é a `4000`.
