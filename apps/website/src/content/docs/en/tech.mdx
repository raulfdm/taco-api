---
title: "Technologies"
---

- [NodeJS](https://nodejs.org/en) - JavaScript runtime for Prisma to generate files (this is probably temporary)
- [Bun](https://bun.sh/) - JavaScript runtime to run scripts and launch the server
- [pnpm](https://pnpm.io/) - package manager
- [ElysiaJS](https://elysiajs.com/) - HTTP Framework to run with Bun
- [GraphQL Yoga](https://elysiajs.com/plugins/graphql-yoga.html) - GraphQL handler
- [Prisma](https://www.prisma.io/) - Object-Relational Mapping (ORM) in JavaScript to populate the database and define relationships
- [SQLite](https://sqlite.org/) - Mini relational database
- [GraphQL](https://graphql.org/) - Query language for the API
- [Starlight](https://starlight.astro.build/) - Site generator for documentation
